# Minishell

> Proudly made by [Arnaud Biju Duval](https://github.com/ArnaudBD) (abiju-du) and [Th√©o Vogel](https://github.com/theovgl) (tvogel) ! 

This project is about creating a simple shell.

## Features

- History
- Single and Double quotes
- Redirections
  - < and >
  - << and >>
- Pipes
- Environnement variables
- Exit status
- Signals
  - <kbd>Ctrl</kbd> + <kbd>c</kbd>
  - <kbd>Ctrl</kbd> + <kbd>d</kbd>
  - <kbd>Ctrl</kbd> + <kbd>\\</kbd>
- Some builtins
  - echo (working with -n)
  - cd
  - pwd
  - export
  - unset
  - env
  - exit
